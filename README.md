## ğŸ”® About The Project

This project focuses on **customer segmentation for an e-commerce platform**, using real transactional data from the **Olist Brazilian E-commerce dataset**.

The main objectives are to:
- Prepare and clean complex relational datasets
- Perform **exploratory data analysis (EDA)** to understand customer behavior
- Use **SQL queries** to answer business-oriented questions
- Build meaningful **customer segments** based on purchasing behavior

Key topics covered in this project:
- Data cleaning and preprocessing
- Relational data exploration (orders, customers, items, reviews)
- Feature engineering for customer-level analysis
- Statistical analysis and visualization
- Foundations for clustering-based segmentation

The project is implemented entirely in **Jupyter notebooks**, with a strong emphasis on **readability, interpretation, and business insight**.

## ğŸª„ Getting Started

This project is designed to be explored locally using **Jupyter Notebook**.

### ğŸ“˜ Prerequisites

- Python 3.8+
- pip or conda
- Jupyter Notebook or Jupyter Lab

### ğŸ§™ Installation

1. Clone the repository:

       git clone https://github.com/joelle-jnbaptiste/SchoolProject---Ecommerce-customer-segmentation.git

2. Navigate to the project folder:

       cd SchoolProject---Ecommerce-customer-segmentation

3. Install dependencies:

       pip install -r requirements.txt

4. Launch Jupyter Notebook:

       jupyter notebook

## ğŸ“– Usage

- Explore the **input/** folder to see the raw Olist datasets.
- Use the **clean/** folder for processed and cleaned CSV files.
- Open the notebooks in order to follow the analysis:
  - Data loading and cleaning
  - SQL-based business queries
  - Exploratory data analysis
  - Feature construction for segmentation
- Visualizations and conclusions are embedded directly in the notebooks.

This project is intended for:
- Data analysis practice
- SQL + Python integration
- Business-oriented data exploration
- Preparation for clustering and segmentation tasks
